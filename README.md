# Namaadhu — AltStore distribution

AltStore/SideStore source for **Namaadhu**, a Maldives prayer times app.

## Add the source

In AltStore/SideStore → **Sources** → **+**, paste:

```
https://raw.githubusercontent.com/chumputys/namaadhu-dist/main/altstore-source.json
```

App binaries are published as GitHub Releases in this repo. AltStore re-signs
the app with your own Apple ID on install.
